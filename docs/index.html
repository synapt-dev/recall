Memory for AI coding assistants

The difference between a stranger and a collaborator is shared context. Synapt gives your AI assistant persistent memory — so every conversation picks up where the last one left off.

Get started GitHub
$ pip install synapt

Features

Search

Hybrid RRF search

BM25 full-text search fused with 384-dim semantic embeddings via Reciprocal Rank Fusion. Surfaces results that keyword search alone would miss.

Search

Query intent routing

Classifies queries as factual, temporal, debug, decision, aggregation, or procedural. Adjusts recency decay, knowledge boost, and embedding weight automatically.

Memory

Knowledge consolidation

Extracts durable facts from session journals into a knowledge base with lineage tracking, contradiction detection, and confidence scoring.

Memory

Session journal

Rich entries with focus, decisions, accomplishments, and next steps. Auto-extracted at session end, searchable across all sessions.

Memory

Topic clustering

Groups related chunks across sessions via Jaccard token-overlap. LLM-generated summaries for fast context retrieval in concise mode.

Tools

13 MCP tools

Search, journal, reminders, timeline, enrichment, knowledge management. Works with Claude Code and any MCP-compatible assistant.

Local-first

Zero cloud dependencies

SQLite + local embeddings. No API keys, no external services. All data stays on your machine. Works offline.

Search

Adaptive working memory

Access tracking boosts frequently-retrieved chunks. Active topics surface naturally without manual curation.

Tools

Plugin system

Extend with additional MCP tools via Python entry points. Plugins are auto-discovered and loaded at server startup.

How search works

Three retrieval paths run in parallel and merge at the rank level — no score normalization needed.

query | +--------------+ | intent classify | +--------------+ / | \ half_life emb_wt know_boost / | \ +---------+ +----------+ +-------------+ | BM25/FTS5 | | Embedding | | Knowledge | | + decay | | 384-dim | | FTS + embed | +---------+ +----------+ +-------------+ \ / | +------------------+ | | Reciprocal Rank | | | Fusion (k=60) | | +------------------+ | \ / +---------------------------+ | interleave + dedup + boost | +---------------------------+ | +-----------------+ | ranked context | +-----------------+

Quick start

1

Install

pip install synapt
2

Build the index

Discovers Claude Code transcripts automatically.

synapt recall build
3

Add the MCP server

Add to ~/.claude/mcp.json for all projects:

{
  "mcpServers": {
    "synapt": {
      "type": "stdio",
      "command": "synapt",
      "args": ["server"]
    }
  }
}
4

Set up hooks

Auto-archives transcripts and writes journal entries.

synapt recall setup
