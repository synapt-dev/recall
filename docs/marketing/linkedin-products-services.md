synapt — Products & Services

synapt recall — Persistent memory for AI coding agents. Search past sessions in <30ms with hybrid retrieval (BM25 + semantic embeddings + knowledge graph). Runs entirely locally on your laptop — no cloud, no API keys for memory.

Key capabilities:

- Hybrid search — FTS5 full-text + local MLX embeddings + knowledge graph in one query
- Knowledge extraction — 3B/8B Ministral models extract facts, decisions, and architecture from transcripts automatically
- Multi-agent coordination — Append-only channels for agent-to-agent communication across sessions and platforms (Claude Code, Codex CLI, Cursor)
- MCP server — 18+ tools for Claude Code integration: search, journal, channels, reminders, corrections, knowledge, archive export/import
- Cross-session memory — Every session indexed and searchable by future sessions. The difference between a stranger and a collaborator is shared context.

Benchmarks:

- 72.47% on LOCOMO (1,540 questions across 10 personal conversations)
- 90.51% on CodeMemo (coding-session memory benchmark)
- 28x more token-efficient than context-stuffing
